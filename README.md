# tsukihime-unity-gal
这是一个自用的unity的gal制作框架，专为psvita平台开发服务
截图以我开发的unity版本的月姬plus为例
解压成单独的项目即可使用
release已自带插件包fungus以及动画管理，可直接调用
欢迎大家参与到psvita的gal的开发过程中来，哈哈！
